# Cloud Graph

Интерактивное веб-полотно для структурирования мыслей, тем и задач через облака и связи между ними.

## Что уже есть

- свободная светлая доска с сеткой;
- создание облаков с названием;
- задачи внутри каждого облака;
- дедлайны у задач;
- связи между облаками и задачами;
- перетаскивание облаков;
- масштабирование колесом и панорама через `Space + drag`;
- сохранение состояния в `localStorage`.

## Запуск

Можно просто открыть [index.html](C:\Users\bosse\OneDrive\Documents\cloud_working\index.html) в браузере.

Если хочется запускать как локальный сайт:

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

После этого открыть [http://127.0.0.1:4173](http://127.0.0.1:4173).
