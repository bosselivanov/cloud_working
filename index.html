Cloud Graph

Свободное полотно для идей, задач и логических связей между ними.

Как работать

  • Тяни облака мышкой, чтобы раскладывать карту.
  • Добавляй задачи внутри облака через кнопку `+ задача`.
  • Нажми `Связать`, затем выбери вторую сущность.
  • Двойной клик по названию позволяет редактировать текст.
